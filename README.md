Kamgaar is a workforce management platform built for heavy-equipment operators and infrastructure contractors.

We employ operators full-time and deploy them reliably to critical projects, eliminating manpower uncertainty and equipment downtime.

Key Features

Role-based login: Contractor / Worker

Google & Phone OTP authentication (Firebase)

Digital Worker Passport (skills, experience, ratings)

Separate dashboards for workers and contractors

Responsive, clean UI (HTML, CSS, JS)

Tech Stack

Frontend: HTML, CSS, JavaScript

Auth: Firebase Authentication

Hosting: Static / Firebase Hosting

Status

MVP complete

Backend APIs & database coming next
